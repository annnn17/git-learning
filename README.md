# Git Learning Project

**Название проекта:** Git Learning Project
**ФИО студента:** Нарышкина Анна Андреевна
**Группа:** 241

## Описание

Учебный проект для освоения базовых операций Git:
- локальный репозиторий
- работа с удалённым репозиторием GitHub
- ветки и Pull Request
- управление доступами

## Цель проекта

Научиться создавать первый осмысленный коммит с понятным сообщением, отражающим суть внесенных изменений.

## Технологии

- Python 3
- Git 2.50+
- GitHub CLI (gh)
